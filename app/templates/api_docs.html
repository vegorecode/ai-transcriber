WhisperX — API-обёртка

Назад

Базовый URL: /. Все ответы JSON, если не указано иное.

POST /api/transcribe

Создаёт задачу транскрипции. Тело: multipart/form-data с полем file (аудио/видео).

{
  "id": "",
  "status": "queued"
}

Далее опрашивайте GET /api/jobs/<job_id> или общий список GET /api/jobs.

GET /api/jobs

Список задач (последние — сверху).

[
  {
    "id": "...",
    "original_filename": "sample.mp3",
    "status": "queued|processing|finished|failed",
    "created_at": "2024-01-01T00:00:00Z",
    "updated_at": "2024-01-01T00:00:10Z",
    "output_format": "srt",
    "output_path": "/work/out/.srt",
    "download_name": "sample.srt",
    "error": null
  }
]

GET /api/jobs/<job_id>

Детали одной задачи.

GET /download/<job_id>

Скачивание результата, если status=finished. Отдаёт файл как attachment.

GET /download-config/<job_id>

Скачивание снэпшота настроек (TXT) рядом с транскриптом — файл *_config.txt.

GET /api/settings

Текущие настройки движка.

POST /api/settings

Обновление настроек. Тело JSON — подмножество полей из GET /api/settings.

{
  "language": "ru",
  "model": "openai/whisper-large-v3",
  "device": "cuda",
  "compute_type": "float16",
  "batch_size": 2,
  "beam_size": 5,
  "temperature": 0.0,
  "temperature_increment_on_fallback": 0.2,
  "vad_method": "pyannote",
  "vad_onset": 0.30,
  "vad_offset": 0.25,
  "diarize": true,
  "min_speakers": 1,
  "max_speakers": 4,
  "output_format": "srt",
  "length_penalty": 1.1
}

Пример: curl

curl -F file=@audio.mp3 http://localhost:8800/api/transcribe
curl http://localhost:8800/api/jobs
curl -OJ http://localhost:8800/download/<job_id>

Пример: JavaScript (fetch)

const fd = new FormData();
fd.append('file', fileInput.files[0]);
const job = await fetch('/api/transcribe', { method: 'POST', body: fd }).then(r => r.json());
const detail = await fetch(`/api/jobs/${job.id}`).then(r => r.json());
if (detail.status === 'finished') window.location.href = `/download/${job.id}`;
